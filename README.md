# GreenTower Steel — Techno-Economic Model

Techno-economic and emissions model for hydrogen direct-reduced iron
(H2-DRI) + electric arc furnace steel, sited in the Kansas wind
corridor, for wind-turbine tower plate.

## Key findings

- At $4/kg H2, green steel needs only ~$57/t CO2 to beat blast-furnace
  steel on *operating* cost — below today's EU ETS range.
- Capital recovery (~$370/t at 200 kt/yr) is the binding constraint,
  not process economics.
- ~1 Mt/yr with concessionary financing brings this inside today's
  carbon prices.
- The "95% CO2 reduction" claim is a *procurement* outcome: 99% likely
  with contracted clean power, 5% without.

## Limitations

No hourly power matching, no electrolysis water accounting,
order-of-magnitude capital costs, placeholder electricity price.

## Running it

Open the notebook in Colab and Run All. numpy + matplotlib only.
